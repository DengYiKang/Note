# 研究计划书

> 肖豪，樊梦妮，邓懿康

## 研究问题

建立一个简易的分布式搜索引擎，能对给定的关键词进行搜索查询，返回相关结果。

## 研究动机

该项目能够很好地帮助我们理解与结合这门课程所习得的知识，将分布式等思想应用于实践。由于搜索引擎面向的数据量大，建立索引等计算任务繁重，运用分布式等技术能带来很大的提升，其巨大的反馈能促使我们有更大的兴趣去提升和完善。

## 现有方法的缺陷

搜索引擎在整体上有以下缺陷：

+ 现有的搜索引擎仅允许用一组关键词及逻辑运算符组成提问，但关键词检索不能完全满足用户的要求。
+ 现有的搜索引擎无法为单个用户提供有效的个性化服务，大大增加了用户查询有用信息的时间。
+ 每一次检索都是从头开始的检索，不能从原有的查询结果中作进一步的提炼。

从技术层面来讲，当前热门的选择有Elasticsearch和Solr，它们都是在基于全文搜索引擎Apache Lucene基础上开发的。

但两者的优缺点也相较明显：

Elasticsearch仅支持json文件格式，而Solr支持更多格式的数据；Solr在传统的搜索应用中表现好于 Elasticsearch，但在处理实时搜索应用时效率明显低于 Elasticsearch，等等。

## 初步思路

+ 爬取大量数据
+ 对数据进行清洗，分词
+ 对数据建立索引并保存
+ 对输入的关键词进行搜索，提交结果，完成搜索

## 预期结果

该简易的搜索引擎能提供关键词查询，其中爬取数据、清洗数据、分词、建立索引、搜索等任务均布置在分布式系统上运行，有效地提高了效率。
